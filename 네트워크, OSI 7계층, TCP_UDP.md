# 네트워크란?

네트워크(Network)란 컴퓨터, 스마트폰, 서버 등의 장치들이 서로 연결되어 데이터를 주고 받는 것을 말합니다.

# OSI 7계층이란?

OSI 7계층이란 네트워크 통신 과정을 7단계로 나눈 모델입니다.

### 7계층 응용 계층 (Application)

사용자가 직접 사용하는 프로그램이 작동하는 계층입니다.

예시

- 웹 브라우저
- 이메일
- 카카오톡

프로토콜

- HTTP
- HTTPS
- FTP
- DNS

### 6계층 표현 계층 (Presentation)

데이터의 형식을 변환하는 계층입니다.

역할

- 암호화
- 복호화
- 압축
- 인코딩

예시

- JPEG
- MP3
- TLS/SSL

### 5계층 세션 계층 (Session)

통신 연결을 생성하고 유지하는 계층입니다.

역할

- 연결 시작
- 연결 유지
- 연결 종료

예시

- 로그인 세션 유지

### 4계층 전송 계층 (Transport)

데이터를 목적지까지 안전하게 전송하는 계층입니다.

프로토콜

- TCP
- UDP

TCP

- 신뢰성 높음
- 순서 보장
- 오류 확인

UDP

- 빠름
- 순서 보장 안 함

예시

- TCP: 웹사이트, 파일 전송
- UDP: 게임, 실시간 방송
    - 실시간 서비스에 사용

### 3계층 네트워크 계층 (Network)

목적지 IP 주소를 이용해 최적의 경로를 찾는 계층입니다.

역할

- IP 주소 사용
- 경로 선택(라우팅)

장비

- 라우터

프로토콜

- IP
- ICMP

### 2계층 데이터 링크 계층 (Data Link)

같은 네트워크 안에서 데이터를 전달하는 계층입니다.

역할

- MAC 주소 사용
- 오류 검출
- 프레임 전송

장비

- 스위치
- 브리지

### 1계층 물리 계층 (Physical)

실제 전기 신호나 광 신호를 전송하는 계층이다.

역할

- 전기 신호 또는 광 신호 전송

장비

- 랜선(UTP)
- 광케이블
- 허브
- 리피터

# TCP / IP 4계층

### 4계층 응용 계층 (Application)

사용자가 직접 사용하는 프로그램과 네트워크 서비스를 제공하는 계층입니다.

예시

- 웹 브라우저
- 이메일
- 카카오톡

프로토콜

- HTTP
- HTTPS
- FTP
- SMTP
- DNS

역할

- 사용자와 네트워크 사이의 인터페이스 제공

### 3계층 전송 계층 (Transport)

데이터를 목적지까지 안전하게 전달하는 계층입니다.

프로토콜

**TCP**

- 연결 지향
- 신뢰성 보장
- 순서 보장
- 오류 검출 및 재전송

예시

- 웹사이트 접속
- 파일 전송

**UDP**

- 비연결형
- 빠른 전송
- 순서 보장 안 함

예시

- 온라인 게임
- 실시간 스트리밍
- 음성 통화

### 2계층 인터넷 계층 (Internet)

목적지까지 데이터를 전달할 경로를 결정하는 계층입니다.

프로토콜

- IP
- ICMP
- ARP

역할

- IP 주소 사용
- 라우팅 수행

장비

- 라우터

### 1게층 네트워크 접근 계층 (Network Access)

실제 네트워크를 통해 데이터를 전송하는 계층입니다.

역할

- MAC 주소 사용
- 프레임 생성
- 전기 신호 전송

장비

- 스위치
- 허브
- 랜카드(NIC)

매체

- UTP 케이블
- 광케이블
- Wi-Fi
